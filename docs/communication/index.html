Communication

Communication comes first because without it, we can't get anything done.

There are three separate places on this page where it says to email Sophie with some information. Please read to the bottom, aggregate everything, and then send them one email with all the information that we're missing from you in one go.

GitHub

If you do not already have a GitHub account, create one. Ideally choose a username that makes it easy for others to identify you, and remember that your GitHub account is going to be a part of your professional identity for a long time, so choose wisely.

If you provided a GitHub username or link in your UROP application, you should have already received an invitation to join our GitHub organization. Please accept this invite now, if you haven't already.

If you haven't received an invite, or didn't previously have an account, email Sophie with your username, so she can add you to our organization.

Even if you already have a GitHub account, you may wish to sign up for the Student Developer Pack.

Slack

You should have already received an invitation to join the DH Lab Slack workspace.

Once you've joined the workspace, send a message on the #random channel, with the title of your favorite book, piece of music, or visual art, so everyone knows how to reach you there.

Google Account

We use Google Drive and Google Docs extensively in the lab to collaboratively edit documents. Please email Sophie your Google Account username, so we can add you to the DH Lab shared Google Drive folder. We recommend that you request an MIT G Suite for Education account and use that for your labwork, such that it's associated with your kerberos ID.

Email

As stated in in the UROP Contract, we expect you to check your email once per business day and expect that you will respond to messages in a timely manner.

Please email Sophie with a headshot photo so we can put it on the website.
