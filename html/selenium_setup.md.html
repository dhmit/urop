Setting up Selenium testing locally

  1. Make sure you've installed the latest version of the Python requirements, including selenium, by navigating to requirements.txt in PyCharm. If you're missing anything, it should prompt you to install the missing requirements.

  2. You'll need to install ChromeDriver, which is a tool for automated testing in Chrome.

macOS: The easiest thing to do is to install chromedriver using the Homebrew package manager. Go to the Homebrew website and follow the directions for installing Homebrew. Then, run the command brew cask install chromedriver to install ChromeDriver and add it to your PATH.

Windows 10: you'll need to download and add ChromeDriver to your PATH manually.

Create a directory C:\bin

Download chromedriver for Chrome 79 for Windows, unzip it, and save the exe to C:\bin

Open the Control Panel, search for 'environment variables', and click 'Edit the system environment variables'.

On the System Properties window, click 'Environment Variables...'

Under 'System variables' select 'Path' and click 'Edit...'

Add the C:\bin directory to your path by clicking New and adding a new entry to the list.

Once you've installed ChromeDriver, verify it's working by restarting PyCharm and, from the backend directory, running python manage.py test.
